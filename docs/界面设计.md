# 十三、界面设计 (UI Design)

## 13.1 设计理念

界面设计遵循"为什么 → 是什么 → 做什么"的三层次理念。为什么层次对应用户需求，研究人员需要可视化理解 GPU 栅栏同步机制。是什么层次对应界面上展示的内容，包括状态面板、拓扑图、内存热力图、性能指标。做什么层次对应用户能执行的操作，包括配置参数、控制仿真、观察数据。

## 13.2 布局结构

界面整体采用左右分栏布局。左侧区域为主视图区域，包含线程块状态表格、树形拓扑可视化、全局内存存储结构三个核心组件。右侧区域为控制面板，包含仿真控制、配置、性能指标、全局内存、事件日志五个功能模块。顶部设置状态栏，显示系统连接状态、仿真时钟 Tick、栅栏类型、仿真状态等信息。

## 13.3 各模块设计说明

### 13.3.1 状态栏 (Status Bar)

状态栏的设计元素包括连接状态指示器、当前时钟 Tick 显示、栅栏类型标签、仿真状态文字。连接状态指示器通过颜色变化反映系统连接状态，绿色表示已连接，红色表示未连接。当前时钟 Tick 显示仿真进度，每 100ms 更新一次。栅栏类型标签根据配置显示 CENTRALIZED、TREE、STATIC_TREE 等算法类型。仿真状态文字显示 STOPPED、RUNNING、PAUSED、COMPLETED 等运行状态。

界面样式为一行式状态显示：[● 已连接] 时钟: 1234 栅栏: CENTRALIZED 状态: 运行中

### 13.3.2 线程块状态表格 (Block State Table)

线程块状态表格的设计元素包括数据表格、状态徽章、工作量数值、栅栏标记。数据表格展示线程块 ID、状态、工作量、是否在栅栏等信息，实时更新行数据。状态徽章通过着色反映线程块状态，RUNNING 显示绿色，WAITING 显示橙色，FINISHED 显示蓝色，FAILED 显示红色。工作量数值显示每个线程块当前完成的 ticks。栅栏标记指示线程块是否处于等待栅栏状态。

界面样式为表格形式，包含线程块ID、状态、已完成工作量、是否栅栏四列。

### 13.3.3 树形拓扑可视化 (Tree Topology Visualization)

树形拓扑可视化的设计元素包括树形层级结构图、链接状态高亮、节点徽章、Block 叶子节点。树形层级结构图用于展示逻辑结构，仅在树形算法时显示。链接状态高亮反映通信过程，汇聚时显示橙色，释放后绿色渐隐。节点徽章显示 count/limit 比例。Block 叶子节点根据线程块状态着色。

界面样式包含树形拓扑图、链接图例（空闲、通信、释放三种状态）、内部节点显示 count/limit 值和完成标记、叶子节点显示 Block 标识和等待状态。

### 13.3.4 全局内存存储结构 (Memory Structure Panel)

全局内存存储结构的设计元素包括变量表、热度条、存储结构展示。变量表配合热度条进行瓶颈透视，热力着色反映内存访问频率，蓝色表示低频，橙色表示中频，红色表示高频。存储结构展示根据算法类型差异化呈现，Centralized 算法使用计数器，Tree 算法使用邻接表，StaticTree 算法使用数组。

界面样式包含存储结构标题、各节点数据结构（count、sense 等字段）、节点间的关联关系。

### 13.3.5 控制面板 (Control Panel)

控制面板的设计元素包括仿真控制按钮、仿真模式选择器、数值输入框、工作量方差滑块、栅栏类型选择、重置并应用按钮。仿真控制按钮组包含启动、暂停、单步三个操作按钮，控制仿真执行。仿真模式选择器支持 NORMAL 与 FAILURE 模式切换。数值输入框用于配置线程块数、栅栏间隔等参数。工作量方差滑块调节负载波动程度。栅栏类型选择切换算法实现。重置并应用按钮使配置变更即时生效。

界面样式为分组展示，仿真控制区包含三个操作按钮，配置区包含各参数输入控件。

### 13.3.6 性能指标面板 (Metrics Panel)

性能指标面板的设计元素包括 KPI 数值显示、静态展示区域。KPI 数值显示同步次数 sync_count、平均通信次数 avg_communication 等关键性能指标。静态展示区域用于数据导出，便于论文截图。

界面样式为表格或卡片式展示，显示指标名称和对应数值。

### 13.3.7 事件日志 (Event Log)

事件日志的设计元素包括时间戳日志流、BLOCK_FAIL 日志高亮。时间戳日志流自动滚动，最新日志显示在顶部，包含时间戳和事件描述。BLOCK_FAIL 日志使用红色标记突出显示，用于故障诊断。

界面样式为列表形式，每行显示时间戳和事件内容，故障事件添加警告标记。

## 13.4 响应式设计

界面支持三种断点布局调整。当视口宽度大于等于 1200px 时，采用左右分栏布局，主视图占 70%，控制面板占 30%。当视口宽度在 768px 至 1199px 之间时，保持左右分栏但缩小比例。当视口宽度小于 768px 时，切换为上下堆叠布局，控制面板置顶，主视图在下。

## 13.5 键盘快捷键

界面支持四个键盘快捷键用于快速控制。S 键启动仿真，P 键暂停仿真，空格键单步执行，R 键重置仿真。
